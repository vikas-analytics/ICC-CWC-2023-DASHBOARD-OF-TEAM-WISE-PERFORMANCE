# 🏏 ICC Cricket World Cup 2023 – Team Performance Dashboard

## 📌 Project Overview

This project is an interactive Microsoft Excel dashboard that analyzes the performance of teams participating in the ICC Cricket World Cup 2023. The dashboard transforms raw match data into meaningful insights using Excel's data analysis and visualization features, helping users compare team performances throughout the tournament.

---

## 🎯 Objectives

- Analyze team-wise performance in the ICC Cricket World Cup 2023.
- Compare wins, losses, and points across all teams.
- Visualize tournament statistics using interactive charts.
- Build an easy-to-use dashboard for cricket performance analysis.

---

## 📊 Dashboard Features

- Team-wise Performance Analysis
- Total Matches Played
- Wins & Losses
- Points Table Overview
- Net Run Rate (NRR)
- Interactive Slicers
- Dynamic Charts & Graphs
- Clean and User-Friendly Dashboard
- Total score team wise
- Highest score by player from Each team
- Highest wicket taker from the team
- Best Bowling Average
- Best Batting Average
